## Armour

Roll d20 to determine whether you have armour.

| d20 | Armour |
| -- | :----- |
| 1–6 | None |
| 7–15 | Gambeson |
| 16–19 | Cuirass armour |
| 20 | Brigadine armour |

| d20 | Armour |
